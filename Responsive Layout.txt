Design System & Tokens
Color Palette
Primary (Purple): #6366F1 (or similar Indigo-600)

Background: #F8FAFC (Slate-50)

Surface (Cards): #FFFFFF (White)

Text (Heading): #1E293B (Slate-800)

Text (Sub): #64748B (Slate-500)

Status Green (Active): #22C55E

Status Red (Urgent): #EF4444

Typography
Font Family: Inter (as per your preference)

Headings: Semi-bold, 20px - 24px

Body: Regular, 14px - 16px

Labels/Tags: Medium, 10px - 12px (All Caps for tags)

🏗️ Component Breakdown
1. Navigation Bar (Bottom)
Layout: Horizontal Auto Layout, fixed at bottom.

Items: Dashboard, Patients, Encounters, Payments.

Active State: Soft purple background capsule with dark purple text/icon.

2. Stats Grid (Dashboard)
Grid: 2x2 Responsive Grid.

Card Style: White background, 16px Border Radius, subtle 4px blur shadow.

Content: Icon (top left), Label (gray), Value (large bold black).

3. Patient/Encounter Cards
Structure: Vertical Auto Layout with 12px-16px padding.

Header Row: Name/Title on the left, Status Tag on the right.

Metadata Row: Two-column split for "Last Visit" and "Insurance" or "Provider."

Border: 1px solid #E2E8F0 or just a subtle shadow.

💻 Implementation Strategy (Code/Figma)
For Figma:
Auto Layout is Key: Set your main container to Vertical Auto Layout with 24px spacing.

Floating Action Button (FAB): Use an absolute position for that purple + button in the bottom right corner (approx 20px from bottom/right).

Search Bar: Use a 48px height, 12px border radius, and #F1F5F9 background.
